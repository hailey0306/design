Overview

This is a coding project that I started from scratch by using jQuery Mobile framework and google map API. It is a self-confession app inspired by seven sins. All the data are pre-made by using JSON Generator.

PROBLEM STATEMENT

Sin exists, and not only is it wrong but it does evil.

Philosopher Tsang daily examined himself, to guard against his being guilty of any imposition. Franklin worked towards creating 13 Virtues that would allow him to be a better person essentially.

So I believe it is important to make Confessions, sometimes people might feel shame, but then they finish Confession, they leave free, grand, beautiful, forgiven, candid, happy. This is the beauty of Confession!

The GOAL

The weight of sin is heavy and you don't have to carry it! For people don't have time to go to church or not a Christian a mobile app that allows them to make confession whenever they want can help them increase their self-knowledge and correct their bad habits.

Thoughts

I've learned a lot of coding skills by doing this project. Google Map API is the most important part in this project and I used a customized “look" for my map to meet my branding of this app. I also used a powerful tool called JSON generator to get all the default data. To put this app in my portfolio, I also redesigned all the monsters. For the value of this app's functionality, I think although the app can help people to confess, however, it still important to eduated people how to confess and why it can help us live better.
